# $year
Returns the current year.

:::tip

This function uses UTC (+00:00) time zone by default. To change the time zone, use `$time[]` function.

:::

## Syntax
```
$year
```

## Example

### Code
```plain title="!year"
$nomention
Current Year: $year
```
